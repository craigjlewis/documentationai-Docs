## Create Your Project

<Steps>
  <Step title="Go to Dashboard">
    Navigate to your dashboard after logging in.
  </Step>
  <Step title="Click 'Create Project'">
    Find and select the 'Create Project' button at the top right.
  </Step>
  <Step title="Enter Details">
    Fill in the project name and an optional description.
  </Step>
  <Step title="Save">
    Click 'Save' to finish creating your new project.
  </Step>
</Steps>

<Callout kind="tip">
Invite teammates anytime from the Project Settings page.
</Callout>
