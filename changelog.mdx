## Upgrading to the Latest Version

Follow these steps to upgrade your CJL_Org instance safely.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Create a full backup of your documentation spaces.

````bash
cjl-org backup --all-spaces --output=/path/to/backup
````
  </Step>
  <Step title="Update Package" icon="download">
    Install the latest version using your preferred method.

    <CodeGroup tabs="npm,yarn">
````bash
npm install cjl-org@latest
````
````bash
yarn add cjl-org@latest
````
    </CodeGroup>
  </Step>
  <Step title="Run Migrations" icon="settings">
    Apply database migrations if applicable.

````bash
cjl-org migrate --env=production
````
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your CJL_Org service to apply changes.

````bash
systemctl restart cjl-org
````
  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, review the [migration guide](https://docs.example.com/migration) for any manual steps specific to your setup.
</Callout>
