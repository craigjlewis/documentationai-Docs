## Overview

CJL_Org provides a centralized space for your project documentation. You can create, organize, and collaborate on docs effortlessly. It supports version control, search, and team permissions to keep everything accessible and up-to-date.

This platform scales with your projects, from small teams to large enterprises. Start by setting up your first workspace and inviting collaborators.

## Key Benefits

Organizing documentation manually leads to scattered files and lost information. CJL_Org solves this with:

- **Centralized Access**: All docs in one secure location.
- **Team Collaboration**: Real-time editing and comments.
- **Version History**: Track changes and revert easily.
- **Search and Discovery**: Full-text search across all content.

<Callout kind="tip">
  Focus on content creation while CJL_Org handles the organization.
</Callout>

## Quick Start Guide

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.cjl.org/register` with your email.
  </Step>
  <Step title="Set Up Workspace" icon="folder-plus">
    Create your first workspace and invite team members.
  </Step>
  <Step title="Create Document" icon="file-plus">
    Start a new doc and use the editor to add content.

````markdown

Welcome to CJL_Org!

- Organize sections
- Add code blocks
- Embed images
````
